Detailed Description

Use for basic statistics, and the interface is self-evident.

Usage:

Stat stat;
stat << 1.0 << 2.0 << 3.0;
Stat stat2;
stat2 << 3.0 << 5.0 << 6.0;
stat.merge(stat2);
stat.getCount();
stat.getSum();
stat.getMax();
stat.getMin();
stat.getStd();
